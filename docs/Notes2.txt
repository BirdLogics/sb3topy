Files:
config.py - Handles loading the config json and holds settings
download.py - Handles downloading projects
parser.py - Handles parsing the project into Python
sanitizer.py - Handles cleaning values and naming identifiers
specmap.py - Handles the conversion map
unpacker.py - Handles loading a project from a zip

Conversion steps:
1. Unpack project
2. Parse project

Parsing steps:
for each target:
    parse each block

for each block:
    parse fields
    get specmap
    parse inputs
    format specmap
    cast result

to get specmap:
    check opcode
    parse switches
    create prototype maps
    return

to get prototype map:
    get def from Prototypes
